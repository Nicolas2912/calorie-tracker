     │ Key User Workflows & Pain Points (LLM Integration Opportunities):                                                                                     │
     │                                                                                                                                                       │
     │ 1. Food Entry Process (HIGH IMPACT)                                                                                                                   │
     │                                                                                                                                                       │
     │ Current: Users manually search foods, enter amounts, select meal types                                                                                │
     │ Pain Points:                                                                                                                                          │
     │ - Time-consuming manual food lookup and nutrition entry                                                                                               │
     │ - Inconsistent food naming and portion estimation                                                                                                     │
     │ - Limited food database requires frequent custom food creation                                                                                        │
     │                                                                                                                                                       │
     │ LLM Integration Opportunities:                                                                                                                        │
     │ - Natural language food logging: "I had a large apple and 2 slices of whole wheat toast for breakfast"                                                │
     │ - Photo-to-food recognition: Analyze meal photos to suggest foods and portions                                                                        │
     │ - Smart portion estimation: Convert natural language portions to weights/measures                                                                     │
     │ - Automatic nutrition data lookup: Fetch nutrition info from descriptions                                                                             │
     │                                                                                                                                                       │
     │ 2. Food Database Management (HIGH IMPACT)                                                                                                             │
     │                                                                                                                                                       │
     │ Current: Manual food creation with extensive nutrition form fields                                                                                    │
     │ Pain Points:                                                                                                                                          │
     │ - Tedious manual entry of nutrition data for custom foods                                                                                             │
     │ - Inconsistent food naming and categorization                                                                                                         │
     │ - Missing micronutrient data for many foods                                                                                                           │
     │                                                                                                                                                       │
     │ LLM Integration Opportunities:                                                                                                                        │
     │ - Auto-complete nutrition data: LLM suggests complete nutrition profiles from food names                                                              │
     │ - Smart food categorization: Auto-classify foods as supplements vs. regular foods                                                                     │
     │ - Recipe breakdown: Parse recipe text into individual ingredients with portions                                                                       │
     │ - Nutrition data validation: Flag unrealistic nutrition values                                                                                        │
     │                                                                                                                                                       │
     │ 3. Goal Setting & Personalization (MEDIUM IMPACT)                                                                                                     │
     │                                                                                                                                                       │

     │ Pain Points:                                                                                                                                          │
     │ - Generic goal presets may not fit individual needs                                                                                                   │
     │ - No guidance on appropriate calorie/macro targets                                                                                                    │
     │ - Limited personalization based on user profile                                                                                                       │
     │                                                                                                                                                       │
     │ LLM Integration Opportunities:                                                                                                                        │
     │ - Personalized goal recommendations: Based on age, weight, activity level, goals                                                                      │
